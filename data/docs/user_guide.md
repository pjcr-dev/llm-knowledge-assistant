# HyperGizmo 3000 - User Guide

## Getting Started
1. Unbox the HyperGizmo 3000 and connect it to a power source.
2. Press the power button for 3 seconds to start.
3. Connect to Wi-Fi through the touchscreen interface.
4. Download the companion app for full functionality.

## Basic Operations
- **Automation Mode:** Set tasks to run automatically via the app.
- **Manual Mode:** Operate the device directly using the touchscreen.
- **Voice Commands:** Activate AI assistant by saying "Hey HyperGizmo".

## Maintenance
- Clean the device with a dry microfiber cloth.
- Avoid exposure to extreme temperatures.
- Update firmware regularly through the companion app.
